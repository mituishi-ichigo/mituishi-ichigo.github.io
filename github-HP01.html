github-HP01 の手順

以下の手順で GitHub Pages にホームページを公開します。

1. GitHub にログインしてリポジトリを作る

GitHub公式サイト: https://github.com/

GitHub にログインしたら、

メールアドレスを入力　　
　Githubに登録　

という項目に、あなたのメールアドレスを入力して、Githubに登録をクリックしてください。

　Githubに登録
　　　　 をクリック

GitHub に登録したら、次はリポジトリを作ります。

GitHub のトップページで、右上の「＋」マークをクリックして、「New repository」を選択してください。

すると、リポジトリを作成するためのフォームが表示されます。

初めてのリポジトリの場合は
リポジトリを作成する
　という大きなボタンが表示されます。その場合は、まずこのボタンをクリックしてください。

リポジトリの設定

Repository name: あなたのユーザー名.github.io と入力してください（例：mituishi-ichigo.github.io）。

重要: この名前にすることで、GitHub が「これはホームページ用のデータだ」と自動で認識してくれます。

Public / Private: 「Public（公開）」を選んでください。

Initialize this repository: 何もチェックを入れずに、一番下の「Create repository」ボタンを押します。

2. ファイルをアップロードする

リポジトリが作成されると、青い文字で

creating a new file

or

uploading an existing file.

「uploading an existing file」
「既存のファイルをアップロードしてください。」 というリンクが表示されます。

「uploading an existing file」
「既存のファイルをアップロードして」をクリックします。

画像 image_47da77.png にある index.html を、ブラウザの画面上にドラッグ＆ドロップします。

画面下の「Commit changes」という緑色のボタンを押して確定させます。

3. 公開を確認する

アップロードが終わったら、少し（1〜2分ほど）待ちます。

ブラウザのアドレス欄に https://あなたのユーザー名.github.io/ と入力してアクセスしてみてください。

画像 image_47e219.png で確認したあの画面が、インターネット上に表示されれば成功です！

あなたのユーザー名のところには、あなたが GitHub に登録したユーザー名を入れてください（例：mituishi-ichigo.github.io）。
